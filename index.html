There are three types of css-

  • inline
  • internal
  • external

    • Inline css-

    This css style has written in line of targeted element/tag.


    Internal css-

    This css style has written in the same page of the targeted html tag/element. Its should be write on the head of that html page.


    External css-

    This css style has written in the another page as stylesheet. But the stylesheet page should be linked with the html page.

    CSS power-
